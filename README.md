# ScreenShoter
Скриншотер с рисованием на Python/PySide6

## Статус: В разработке 🚧
- ✅ Работает: захват экрана, рисование, сохранение
- 🚧 В процессе: звуки, настройка горячих клавиш,системный трей ,отмена/повтор действий при рисовании,отчиска полотна
- 📅 Планы: поддержка Linux,

- ## ✨ Возможности

### 🎯 Захват экрана
- Выделение любой области с затемнением фона
- Работа через горячие клавиши (Ctrl+Shift)

### 🎨 Редактирование
- **Перо** — рисование линий с настраиваемой толщиной
- **Маркер** — полупрозрачное выделение  с настраиваемой толщиной 
- **Перемещение** — панель инструментов можно двигать

### 💾 Сохранение
- **В буфер обмена** — мгновенная вставка в любой редактор
- **На диск** — автоматическое именование по дате (screenshot_20240220_153045.png)
- Выбор пользовательской папки для сохранения

### ⚙️ Настройки (сохраняются в реестре)
- Цвет пера и маркера
- Толщина линий
- Путь для сохранения

Установка одной командой:
pip install -r requirements.txt
