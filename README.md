## ✨ Возможности

### 🚀 **Базовый функционал**
- ✅ **Захват любой области экрана** — просто выдели мышкой нужную область
- ✅ **Горячие клавиши** — мгновенный вызов по `Ctrl+Shift` (настраивается)
- ✅ **Системный трей** — приложение не мешает, всегда под рукой

### 🎨 **Редактирование скриншота**
- ✏️ **Перо** — рисуй линии, указывай на детали (цвет настраивается)
- 🟡 **Маркер** — полупрозрачное выделение важных фрагментов
- 🖌️ **Настраиваемая толщина** — от тонких линий до жирных выделений
- ↩️ **Отмена действий** — удаление предыдущих линий (по 5 за раз)

### 💾 **Сохранение результатов**
- 📋 **В буфер обмена** — Ctrl+V в любой редактор или мессенджер
- 💽 **На диск** — автоматическое сохранение в формате PNG
- 📁 **Выбор папки** — сохраняй куда удобно, путь запоминается
- 📅 **Умное именование** — `screenshot_20240220_153045.png`

### ⚙️ **Настройки (сохраняются в реестре)**
- 🎨 Цвет пера и маркера
- 📏 Толщина линий
- 📂 Путь для сохранения по умолчанию

## 🚀 Быстрый старт

### Установка за 30 секунд:

```bash
# Клонируем репозиторий
git clone https://github.com/serega7222/screen.git
cd screen

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем!
python main.py


Как пользоваться:
Запусти программу — появится главное окно

Нажми Ctrl+Shift — экран затемнится

Выдели область — зажми левую кнопку мыши

Рисуй! — используй перо или маркер

Сохрани — в буфер или на диск

🏗️ Архитектура проекта

Проект построен на паттерне MVC (Model-View-Controller):
📁 screen/
├── 📁 controllers/    # Бизнес-логика (нажатия, сохранение, рисование)
├── 📁 model/          # Работа с данными и реестром
├── 📁 view/           # Интерфейсы (главное окно, выделение, рисование)
├── 📁 utils/          # Логирование и утилиты
├── 📁 static/         # Стили и иконки
└── main.py           # Точка входа
